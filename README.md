## Run Demo
페이스북, 카카오 등의 소셜 공유 기능을 사용하기 위해서는 각 서비스의 공유 기능 관련 설정 후 appId를 얻는다.<br><br>
`.env.development` 파일을 참고하여 `.env` 파일을 생성 후, `FACEBOOK_APPID`, `KAKAO_APPID` 항목에 각각 값을 입력하고 `npm start` 커맨드로 웹 서비스 실행.
